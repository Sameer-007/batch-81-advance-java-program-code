An interconnected system of public webpages accessible through the Internet.

the Web is one of many applications built on top of the Internet.

The HTTP protocol governs data transfer between a server and a client.

To access a Web component, a client supplies a unique universal identifier, called a URL (uniform resource locator) or URI (uniform resource identifier)

HTML (hypertext markup language) is the most common format for publishing web documents.
HTML (HyperText Markup Language) is a descriptive language that specifies webpage structure.

An HTML document is a plaintext document structured with elements. Elements are surrounded by matching opening and closing tags. Each tag begins and ends with angle brackets.
